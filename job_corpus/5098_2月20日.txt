大學主要學美術(油畫、水彩、國畫等)、平面設計(ai、ps)、3D(sketch up但很淺的使用)
會做簡單動圖，修片合成也沒問題，手繪能力有科班出來的程度，

目前畢業3年，今年滿25歲~

工作近3年心得 :
第一份工作 : 小型設計廣告工作室
公司人數 : 5人(含老闆)
工作內容 : LOGO、名片、DM、包裝設計...等，平面設計都接
工作薪水 : 23000 (最底薪)
工作地點 : 嘉義
工作時間 : 1個月
心得 : 公司很小，所以相對來說沒什麼制度，但對於剛畢業的我實在能學習實戰設計很多，例如公司長期配合的茶葉商，每季都要做包裝設計，因為從小學開始學美術，所以我的手繪有一定的人能力，對於設計教文青的案子都蠻順利的，但公司在南部想必設計業也知道，大部分業主都喜歡比較俗的東西，通常做兩個版本，業主都選我覺得不好看的。
這份工作只有一個月的原因如下: 
1.公司薪水發現金，等於沒有薪轉，我覺得對於我是個困擾，等於你沒有收入的概念，要先資證明要特地跟老闆要。
2.公司前輩說23000，2年內是不會再調整的，他當時也是做到第三年才從23000變成25000。
3.老闆會帶我去處理他個人接的業務，其餘小事情就是沒有制度、老闆情緒化，到最後公司含我，3個設計師都離職 !

第二份工作 : 飯店美編 
公司人數 : 80人左右
工作內容 : 美編設計 (海報、節慶設計、粉絲團管理...等)
工作薪水 : 25000 (未扣勞健保)
工作地點 : 嘉義
工作時間 : 6個月
心得 : 飯店有一定的制度比較完善，雖然薪水不高，但工作內容不難，飯店人員也較為年輕都好相處，工作內容比較雜，你能想到飯店所有的設計都是，例如宴席海報、節慶禮盒、宣傳行銷等，但也算上手試用期也提早通過，這時就學了一點行銷，觀察其他飯店學習、也能做平面跟包裝設計，算是蠻喜歡的工作，該給的福利都有，比較麻煩是排班，不跟著紅字，但美編也沒什麼要輪班，所以我還是都排周休，當時想若沒意外，會做個2.3年再看是否要更精進，但當時家裡出了一些意外，回了北部工作，算是無奈離職。

第三份工作 : 連鎖飲料美編
公司人數 : 30人左右
工作內容 : 美編設計 (海報、節慶設計、粉絲團管理...等)
工作薪水 : 27000 (未扣勞健保)
工作地點 : 基隆
工作時間 : 1年
心得 : 先說工作內容包山包海的設計，包裝、節慶、行銷活動、小編、所有平面設計，我是不介意反正都很順手，這時也有學到一點構想行銷活動，門市實際狀況，然後分析再修改，雜事很多，還幫忙過會計作帳、出口細節等，很在意的缺點 : 沒有見紅休只有周休，沒有三節禮金，沒有年終，就是算不太有完整制度的公司，但工作內容方面都還好，當時是因為有家人生病，所以我放棄汐止一家代理公司，那間代理公司規模還不錯，代理商品是全省家樂福、寶雅、全聯都有銷售，往返醫院再上班相對來說很方便，可以晚上睡醫院早上再騎車去上班，當時就決定做1年，之後也不用跑醫院了就離職。

以上三份工作，我都抱著我年輕所以都去嘗試工作 ~ 所以評估一下想離職我就會離職，我當時也在想是不是太草莓，但如果真的不好何必繼續浪費時間，我在求職過程中，也沒有找不到工作的狀態，當然工作好壞就很難講，水平也較低，但我很慶幸自己怎麼找還是再設計打轉，平常也會接案子，算小的LOGO、平面設計等~ 也愛插畫 !

第四份工作 : 電商小品牌
公司人數 : 7人左右
工作內容 : 行銷美編設計 (官網行銷以及所有平面、包裝設計)
工作薪水 : 30000 (未扣勞健保)
工作地點 : 汐止
工作時間 : 目前9個月
心得 : 我個人很喜歡小公司因為真的學的東西比較多，這間公司的老闆會直接帶我做行銷、SEO關鍵字、google後台分析...等，這個我真的學得比較細，我可以直接從後台看每個消費者的購物流程，他點了什麼看了什麼我都能抓到數據，然後去分析甚麼活動商品比較吸引人，也能從我發的行銷廣告抓有多少人點，算比較有專業上分析數據的行銷工作，設計方面就是商品文案、banner、包裝設計，這些本身很上手，老闆評價也高，但我行銷還是很弱，嘗試很多但都沒有實質達到消費力提升，就一直不段嘗試，投放廣告找部落客..等等，還能跟momo之類大平台的負責人請教活動，跟提報活動，目前心想可以做個2.3年，覺得電商也算一個大商機，但不能否認薪水還是不高，慢慢進步也算個安逸的工作，公司該給的有給。

有想過往台北、松山、南港等，下一份工作會直接試試看 !
