小弟現在大三升大四，在這段期間，可以看到有好多身邊的同學、朋友不是開始實習、就是去參加商業競賽、個案競賽，每個履歷寫出來都超級厲害，但我在大三的時候履歷上就是一個在台東當翻譯的經驗，看著身邊的朋朋各個起飛，發現自己只是一隻菜雞。

之前爬過很多文，發現以商科來說，競賽、實習是讓你起薪提升、實力拉高的方法，但究竟哪個是最好的，我還沒看過有人比較他們的差異，因此在大三下，頂著學校成本管理會計和消費者行為的兩大壓力下，我決定都親自體驗看看，商業競賽和企業實習究竟哪個比較好。

先看這個學期的成果：
ATCC某企業隊當中的第2名 ( 差1名進決賽… )
和泰汽車售後行銷部實習生 ( 我知道版上心得很多，但我是主要比較差異，plz嘴小力QQ )

商業競賽和企業實習的共同特點就是都在解決問題，那究竟在解決問題上，會有什麼差異呢？以下分為幾個面向來討論商業競賽和企業實習：

1. 問題解決實務面 – 你在解決問題的時候會遇到什麼狀況？
商業競賽你會和同學、或是認識的一起比賽，任何決定、任何疑問你都可以直接迅速的討論，就算這個問題有多笨，你頂多被你同學扒頭，然後得到答案；或是可以立馬進行討論。以我自己的經驗，在比賽期間，有任何問題我們都直接線上或是當場討論，在發現別人或自己的做錯的地方，也都會以幽默的方式，互相指教。但當然很多問題會遇到沒辦法解決的狀態，因為大家的背景、實力可能差不多，因此，這也就是找到對的指導教授的好處，除了自己組內討論完，和教授討論也很重要。

企業實習裡，你附近的人都是你的同事，大家都有自己的事情要忙 ( 當然也有團隊合作類型的實習，但以個人專案較多的暑期實習來說 )，而且大部分的人工作內容、背景都差滿多的，你如果需要別人的幫忙，你會需要花一點心力和時間跟別人解釋你的問題背景，或是等他們有空的時候才有辦法討論，所以通常需要你真正思考過這個問題的重要性，你才會勇敢提出這個問題。大多時候，跟同事討論，可以獲得的東西可能比你和同學討論來的多，因為同事們可以不同背景、或是更實務的經驗，分享他看事情的角度，觀點可能相較跟背景相同的同學討論來的更不同。以本身在和泰汽車的經驗，我曾經請教關於活動舉辦的問題，技術背景出身的同事，就能指出一些你平常完全不會想到的技術問題。

2. 人際面 – 跟你一起解決問題的人，會有哪些狀況？
商業競賽因為它的特性，短時間內不斷要產出有品質的企畫書，你會有一種在船上和大家共患難的感覺，一起熬夜、一起緊張、一起抱怨簡直是日常，所以通常比賽完，大家的感情都會變得比較好，變得比較熟，你會獲得一群很棒的朋友。 ( 這當然建立在隊友的投入是相符之下，所以選隊友很重要 )

企業實習裡，就等於是認識一群新朋友，一樣會有各種新環境的問題，但當然因為你是實習生所以大多數的人會比較照顧你；大多數企業，像和泰汽車會有Mentor帶，或是下班後同事會揪你打球、活動，你會跟大家熟得比較快，時間久了就像家人一樣，所以這就是你能不能適應新環境的問題了。

3. 資源獲得面 – 解決問題中、解決問題後，你有什麼資源？
商業競賽的資源真的比較少，因為你們團隊就是真的自己一個團隊，所有的東西都是靠自己來，你要找廠商合作、找資源，都要靠自己一步一腳印，甚至有時候會被懷疑；我的經驗是就算到了最後一階段 ( 總決賽之前的實作 )，企業分配的導師也是偶爾提供你一些協助 ( 據他們的說法是讓你成長，我是認同啦 )。最重要的是，沒有總決賽就沒有獎金！以金源上來說真的比較少。

企業實習就幾乎等於工作，資源會真的比較多，你要市調資料作分析、廠商合作資源，稍微找一下馬上就有，以我在和泰汽車的經驗，我需要製作一份美宣品，我甚至只要做一份簡單的sample丟給合作廣告商，你的東西就解決了。最重要的是，你雖然每天會過得很累，但發薪水的時候，看到銀行戶頭金額增加，心中會有一份真正的踏實感 ( 無薪實習真的是…辛苦了你們 )

4. 文化類 – 解決問題當中，你會怎麼跟企業文化互動？
商業競賽，你跟企業的文化的互動，是需要去猜想對方的企業文化下想要的是什麼？以我們那組的企業來說，他們在想的東西是無邊無際的，甚至超出題目本身 ( 我們第2名的原因… 想的不夠大 )。但若是在團隊中，你會需要的是一個共同的文化、語言、討論步調、共同記憶，讓大家能維持士氣、順暢溝通，建立自己的文化，這點是非常有趣，也是非常特別的一趟旅程，如何從朋友變戰友，值得大家去探索。

企業實習，你跟企業文化的互動，是你就要成為那個文化下的人類；這就是企業實習的好處，每間企業的文化都不同，透過實習來篩選產業、公司特性，可以知道自己以後想要外商的開放快速，還是台商的熱情、謹慎。以和泰來說，企業充滿日式+台式的感覺，我在這期間學會很多可能是日商的邏輯上嚴謹思考方式，也體驗到台商那種熱情、大家一起來的感覺 ( 自行體會，難以言喻 )。總體來說，就算實習沒有太大的工作成長，你在文化上的體驗絕對是不會少的。

5. 其他類 – 心情篇
商業競賽中，你的專案究竟在評審眼中如何，其實滿難知道的，可能是今年賽制的關係，評審從初賽就喜歡我們的提案是企業隊決賽當天才知道，在這期間我們不斷害怕自己的提案不如別人，心情上上下下，就像是坐海盜船。當然除了製作提案期間的心情，企業隊決賽的最後一天，我們有機會和企業的高層 ( 總經理 ) 吃高級餐廳，其實心情是不錯的，聽他分享怎麼看待職涯等，這部分其實是愉悅的。

企業實習中，你的專案在主管眼中如何，真的非常容易了解，因為做不好、哪裡有錯，主管都會直接地告訴你，當然不是破頭大罵，但這種直接、快速的回饋其實非常有助於能力提升。以我在和泰的經驗，我曾經一份案子過了5次，Mentor、副理、經理，他們都提出了建議，雖然不斷修改考驗著挫折忍受力，但提報時拿出一份被終於被認可的專案，少了些愉悅，多了些踏實感。

6. 給大家的小小建議
會寫這篇是因為在大一二的迷惘期間，從Dcard、PPT獲益良多，因此找個大家沒有討論過，但可能會有的疑問，來以實際經驗跟大家分享回饋社會；希望看過這篇文章，你能更了解自己未來想要嘗試哪條路，畢竟身為商科的學生我們都在摸索前面漆黑的道路。如果你才大一二，請好好上課、或是自己學習，找到自己有興趣的領域好好鑽；如果你大三大四，可以思考一下如何在工作前累積至少一份讓面試官可以更認識你如何解決問題、面對困難的經歷，祝各位迷惘的商科學生都能找到自己的出路！
