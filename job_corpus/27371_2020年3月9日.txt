開學的時候，想說從大一開始存學貸，於是找了一份飲料店的打工。想說時間能配合上，發個試喝而已沒什麼，殊不知，事情沒有我想像的這麼簡單😀


第一天他要我先從打烊班 ( 21.~24. ) 學起 
我第一天收到24.半才下班 
第二天22.~25. ( 凌晨一點) 不知道我幾點才下班
 ( 比原本提早半小時收班 結果我還是收到凌晨一點 ) 
第三天17：30~20：30 ( 結果我還是21.下班 ) 
只有這天發了整整三個小時的試喝，然後去洗刷

你知道我上班都在做什麼嗎？
整整兩個小時都在發試喝
一盤大約都18杯左右 
我每天至少發整整9盤，手直接僵掉
然後還要洗一堆雜七雜八的東西 
什麼量杯、湯匙、雪克杯
榨汁機、果汁機、奶精機
封膜機、盤子、桶子
然後吧臺 ( 就做飲料的地方 ) 總共有三個，昨天我前面一個班的同事，已經幫我先洗好一個了
吧臺是可以拆得都要拆下來洗，然後要刷要擦，所有的櫃子跟櫃面跟吧臺都要擦，那是因為有人幫我，所以洗了1個半小時而已，金價靠北。

以下我那時候面試談的

中班 18：00~21：00
晚班 21：00~24：00

一級 發送飲料、收銀

✔黑色上衣 ( 長短袖、外套皆可 ) 
✔校褲、長褲、裙子
✔布鞋 ( 上班舒服即可 ) 
❎睡褲、運動褲、海灘褲、短裙 

然後我 
沒有勞健保ㄛ
基本時薪都158了= = 

禮拜五第一天，禮拜六第二天，我連兩天打烊班 
禮拜六下午還打電話問我，適應得可以嗎？
：嗯…有點累、下班有點晚
這些我們一開始面試的時候就有談過，多做幾次身體適應就會上手了。
：沒事，我在試試看幾天吧！ 
然後就告訴我今天幾點上下班
22.~25.(凌晨一點) 

我一開始就是因為不想做夜班 
不想做超過22.的工作才選它的 
喔對 他還有說
但是基本上一個禮拜是兩天打烊班，排假日的話，應該足夠休息啦。
 ( 喂，不是這樣的吧 

他排班的原則是：
做一休一、假日至少一天
所以如果我這禮拜排到3天 就是有2天打烊  =66%機率要超過12.下班
至於如果我這禮拜排到4天 然後有2天打烊  =50%的機率要超過12.下班

這完全違反了我上班的初衷好嗎
